# HomePage Project

This is my homework project, available here: (https://shirac631.github.io/homepage/).

## Project Overview

The project consists of an `index.html` page that contains links to different pages demonstrating various CSS approaches:

- Plain CSS – a page styled using only basic CSS.
- CSS generated with Copilot – a page where CSS was assisted/generated by GitHub Copilot.
- Flexbox page – a page demonstrating layout using Flexbox.

## Purpose

The goal of this project is to showcase responsive web design techniques and compare different methods of styling pages with CSS.

## How to View

1. Clone the repository: `git clone <your-repo-url>`
2. Navigate to the homepage directory: `cd homepage`
3. Open `index.html` in your web browser.
